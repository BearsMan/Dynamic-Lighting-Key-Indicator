# Dynamic Lighting Key Indicator

Uses the Windows 11 "Dynamic Lighting" API to change the color of the **`Num Lock`**, **`Caps Lock`**, and **`Scroll Lock`** keyboard keys based on their toggle status.

Requirements:
- Windows 11 (23H2)
- A compatible keyboard
